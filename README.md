# RGACompare

A program for plotting multiple RGASoft .rgadata scan files for easy comparison

# Installation

Clone repository and run the ```main.py``` to start the program

```sh
# Install dependencies
pip install numpy pyqtgraph PySide6

```
requirments are also in '``requirments.txt``'
