A program for plotting multiple RGASoft .rgadata scan files for easy comparison, helpful assessing vacuum conditions over time.

# Installation

Clone or download the repository and install the dependencies.

```sh
# Install dependencies
pip install numpy pyqtgraph PySide6 qt-themes
```
Dependencies can also be installed using the '``requirements.txt``' file.
```sh
# Install dependencies
pip install -r requirements.txt
```
Program can be excecuted by going into 'rga_compare' in terminal and running:
```sh
python -m main
```
